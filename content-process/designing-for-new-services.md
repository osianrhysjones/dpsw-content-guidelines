# Designing content for a new product or service

## Steps

1. Start by understanding the needs of your users and any relevant business goals
2. If you’re publishing on the CDPS website, check if any content types meet this existing need. Contact Alun Bowen/website team to discuss this if you’re unsure
3. If you think you’ll need a new content type or a significant change of use, you’ll need more support from Alun/website team as well as an interaction designer. This might need more resource
4. Prototype some content that meets the needs of your users and business goals. You might want to do this using Google Docs or Microsoft Word
5. Consider how it will be created bilingually: it would be best to trio write and trio edit when doing initial design work for a new product or service
6. Test this content with users
7. Prototype and test on the CDPS staging site
8. Work with Alun/website team to get this live

## Who can do this

Contact a content designer to help with this.
